# Hotel Reservation System

```bash
git clone https://github.com/volodymyrprokopyuk/hotel-reservation.git
cd hotel-reservation
python -m venv env
source env/bin/activate
pip install -r requirements.txt
./bin/deploy.sh
deactivate
```
